🏠 Home Assistant

How HA integration works

  • Enter your HA URL and a long-lived access token in the Device UI → Home Assistant panel.
  • The ESP32 polls GET /api/states/{entity_id} every 30 s — no cloud, no HACS, pure LAN.
  • Up to 4 sensors shown on the HA tab (temp, humidity, any entity). Value + unit displayed automatically.
  • Up to 4 controls (switch / light / scene / script) — tap on screen to toggle or trigger.
  • To get a token: HA → Profile → Long-Lived Access Tokens → Create token.

🚀 Getting started

  • Fill in your WiFi credentials and location below — config saves automatically to your browser.
  • Click Download secrets.h in the bar below and place the file in include/secrets.h of the cloned repo.
  • Run python configurator.py locally to compile and flash via USB or OTA.
  • Clone: git clone https://github.com/RolfKoenders/Deskbuddy.git
